# Rendering JSX

Bu repository sunumda gördüğünüz kodları içermektedir. Bu kodları çalıştırmak için aşağıdaki adımları takip edebilirsiniz.

Tarayicida goruntulemek icin:

```
npx parcel index.html
```

Build almak icin:

```
npx parcel build index.html
```
